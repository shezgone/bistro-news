# 뉴스 인과관계 분석 — 기획 작업 요약

> 작성일: 2026-06-17 · 거시경제 모델용 뉴스 인과관계 분석 프로젝트

## 한 줄 목적
**뉴스 기사에서 경제 사건을 뽑아 "무엇이 무엇을 일으켰나"를 그래프로 엮고, 이걸 숫자(지수)로 바꿔 거시경제 예측 모델에 넣는다.**

## 파이프라인 (5단계)

| 단계 | 하는 일 | 비유 |
|---|---|---|
| **1. 수집·중복제거** | 기사 모으고 같은/비슷한 기사 합치기 (SHA256 해시 + BGE-M3 임베딩) | 같은 사건 기사 한 묶음으로 |
| **2. 이벤트 추출** | LLM이 기사에서 핵심사건·배경·결과를 5W1H로 뽑음 | 기사를 카드 한 장으로 요약 |
| **3. 클러스터링** | 같은 사건끼리 다시 묶기 | 비슷한 카드끼리 분류 |
| **4. 인과 그래프** | 사건들을 "원인→결과" 화살표로 연결 | 사건 지도 그리기 |
| **5. 지수화→예측** | 그래프를 일별 숫자로 압축해 경제지표와 비교 | 지도를 온도계 눈금으로 |

## 핵심 실험 ① — 이벤트 추출 프롬프트 (`뉴스 인과관계 분석.xlsx`)

기사를 사건 카드로 바꾸는 게 전체 품질을 좌우. 그래서 두 방식을 비교:

- **기존**: 프롬프트 1개로 한 번에 다 뽑기
- **신규**: 4단계로 쪼개기 → **A**(핵심사건 1개) → **B**(배경·원인) → **C**(결과·전망) → **D**(품질검증·교정)
  - 장점: 단계마다 규칙이 단순해져 정확도↑ (특히 "익명 관계자 = 사실 아님" 같은 출처 판별을 먼저 적용)

**비교한 모델**: HCX-007 / HCX 32B / Claude Sonnet 4.6 / Gemini 3.1 Pro
**숨은 목표**: 저렴한 국산 **HCX 32B**를 신규 4단계로 돌려 **Claude 수준**까지 끌어올리기
**⚠️ 한계**: 아직 기사 **2건**(호르무즈 통행료 / 범정부 TF) 정성 비교뿐 — 점수화된 벤치마크 없음

## 핵심 실험 ② — 인과 그래프 (3단계 필터)

가짜 인과를 걸러내는 게 관건:
1. **시간 순서 강제** — 원인이 결과보다 먼저여야 함 (역방향 차단)
2. **가중치 계산** — 시간 선행성 × 의미 유사도 × 보도 빈도
3. **이중 검증** — LLM 재검토 + PCMCI 통계검증

이후 **그래프 중심성(eigenvector centrality)** 변화량을 일별 지수로 만들어(= Track B), Granger 검정으로 "이 지수가 경제를 실제로 예측하나" 확인.

## 근거 논문 (검증 완료 ✅)
**Tilly & Livan (UCL), 2021** — *Macroeconomic forecasting with statistically validated knowledge graphs* (arXiv:2104.10457, Expert Systems with Applications 게재).
뉴스 그래프의 중심성 변화로 3개 대형 경제권 **산업생산 예측력 개선**. → 기획팀 설계(Track B + 통계검증)의 직접 근거.

## 데이터 수집 — 한국언론진흥재단 빅카인즈 (BIGKinds)

- **빅카인즈란**: KPF가 104개 언론사 뉴스 ~1.1억 건을 모아 제공하는 공식 뉴스 빅데이터 플랫폼. 공공·연구기관용 **합법 수집 경로**.
- **⚠️ 최근 변화(정황)**: 무료 오픈 API가 **특정 프로그램(해커톤·지원사업 선정사)으로 제한**되고, 일반 이용은 **유상 라이선스**로 바뀐 것으로 보임. (정확한 2026년 요금은 미확인 → **KPF 뉴스빅데이터팀 02-2001-7574** 직접 확인 필요)
- **대안**:
  | 경로 | 본문 | 비용 | 적합도 |
  |---|---|---|---|
  | 빅카인즈 유료 기관계약 | ○ | 유상 | 가장 합법·국내정합 (1순위) |
  | 연합뉴스 직접 데이터계약 | ○ | 유상 | 기관 직접계약 용이 |
  | 공공데이터포털 KPF 데이터셋 | ✕(메타) | 무료 | 보조용 |
  | GDELT (글로벌) | △ | 무료 | baseline·해외이슈·교차검증 |
  | RavenPack / LSEG | ○ | 고가 | 금융특화 상위 비교군 |
- **저작권 핵심**: 한국 기사는 대부분 저작권 보호 대상(단순 사실전달만 예외, 저작권법 §7). 안전 설계 = **라이선스된 소스 사용 + 본문은 해시·임베딩 후 폐기, 추출된 구조화 사실만 보존**(copyright-by-design). 외부 LLM 전송 시 라이선스 범위·역외전송 별도 확인 필요.
